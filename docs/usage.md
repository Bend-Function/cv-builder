# 使用指南

本文档面向最终用户，介绍如何使用 CV Builder 编辑和导出简历。

## 快速开始

打开应用后，左侧是编辑器，右侧是 A4 预览。所有更改都会**自动保存**到浏览器本地存储。

## 编辑简历内容

### 联系信息

- **全名** — 显示在简历顶部，大号字体
- **城市** — 如"上海，中国"
- **电话** — 如"138-0000-0000"
- **邮箱** — 如"yourname@email.com"
- **LinkedIn** — 如"linkedin.com/in/yourname"
- **GitHub** — 如"github.com/yourname"
- **作品集** — 如"yourname.dev"

> 邮箱、LinkedIn、GitHub、作品集链接在导出的 PDF 中是**可点击的超链接**。

### 职业简介

支持两种形式，通过设置面板中的开关切换：

- **一段话** — 写一段完整的职业概述
- **要点列表** — 用 bullet points 列出核心优势

切换时数据不会丢失，方便随时尝试不同形式。

### 技术技能

每行一个技能类别：
- **类别名** — 如"编程语言"
- **技能项** — 如"JavaScript, TypeScript, Python"（用逗号分隔）

渲染时显示为：**编程语言**：JavaScript, TypeScript, Python

### 工作经历

每个经历条目包含：
- 职位名称
- 公司名称
- 地点（可选）
- 起止日期
- 成就要点（bullet points）

点击"添加要点"可以增加 bullet，每个 bullet 对应一行成就描述。

### 项目经历

与工作经历结构相同，用于展示个人项目、开源贡献或学校项目。

### 教育背景

- 学位、学校、地点、起止日期
- 详细信息（可选 bullet points：如荣誉、GPA、主修方向）

### 证书

- 证书名称、颁发机构、获得日期

### 推荐人

三种模式：
- **省略** — 整个推荐人章节不显示
- **应要求提供** — 显示一行"Available upon request"
- **完整列表** — 显示每位推荐人的姓名、职位、公司和联系方式

## 章节管理

### 开关章节

每个章节卡片右上角有一个开关。关闭后该章节在预览和 PDF 中隐藏，但数据仍被保存。

### 拖拽排序

按住章节卡片左侧的拖拽手柄（六圆点图标），可以上下拖拽调整章节顺序。

### 折叠/展开

点击章节标题可以折叠或展开该章节的表单。同时只展开你正在编辑的章节，可以让界面更清爽。

## 调整外观

点击工具栏右上角的**齿轮图标**打开设置面板。

### 预设

预设是"主题 + 版式"的快照：
- 从下拉菜单选择预设，一键切换整套外观
- 点击"保存"按钮，输入名称，将当前设置保存为新预设
- 自定义预设可以重命名或删除（默认预设受保护）

### 主题

四种主题可选：
- **Classic Blue** — 经典商务风，海军蓝章节标题
- **Crimson Block** — 醒目深红色块标题
- **Minimal Mono** — 极简衬线风格
- **Functional** — 功能主义，两栏工作经历，无衬线字体

### 版式

**边距（毫米）**
- 上、下、左、右四个边距均可独立调整
- 默认值：上下 15mm，左右 20mm

**间距（磅）**
- 章节间距 — 相邻章节之间的距离
- 标题间距 — 章节标题与内容之间的距离

**字体**
- 正文字体 — 影响所有正文内容
- 标题字体 — 影响章节标题和姓名

> 可选字体均为系统安全字体，确保在任何环境下都能正确渲染：Arial、Times New Roman、Georgia、Courier New、Verdana、Trebuchet MS

**字号（磅）**
- 正文字号 — 默认 10.5pt
- 标题字号 — 默认 11pt
- 姓名字号 — 默认 22pt

所有版式更改**实时反映在预览中**，无需刷新页面。

## 导出

### 导出 PDF

点击工具栏的**下载 PDF**按钮：
- 服务端生成 PDF，保持预览中的字体、边距和主题
- 文件名为 `resume.pdf`
- 多页简历会自动分页，每页都有正确的边距

### 预览 PDF

点击**预览**按钮，会在新标签页打开 PDF，方便在打印前检查效果。

### 导出 JSON

点击**导出 JSON**按钮下载 `resume-YYYY-MM-DD.json`：
- 这是你的完整简历数据，包括所有内容和版式设置
- 可以用于备份，或迁移到另一台设备

### 导入 JSON

点击**导入 JSON**按钮选择 `.json` 文件：
- 会完全替换当前编辑器中的内容
- 如果文件格式不正确，会提示"Invalid resume file format"
- 导入的旧版本数据会自动迁移到新格式（如旧版的 `summary` 字段会自动转为 `profile`）

## 数据安全

- 所有数据保存在**浏览器本地**（`localStorage`），不会上传到任何服务器
- 清除浏览器数据会丢失简历，请定期**导出 JSON 备份**
- 使用隐身/无痕模式时，关闭窗口后数据会丢失

## 常见问题

**Q: 预览看起来正常，但 PDF 中字体不对？**
A: 确保选择的字体是系统安全字体（设置面板中的下拉选项）。自定义字体在服务端 Chromium 中可能不存在。

**Q: PDF 中的链接点不开？**
A: 请使用 Adobe Acrobat Reader、macOS 预览、或浏览器内置 PDF 查看器打开。某些旧版 PDF 阅读器可能不支持链接。

**Q: 拖拽排序不生效？**
A: 确保你拖拽的是左侧的六圆点图标，而不是整个卡片。如果仍有问题，尝试刷新页面。

**Q: 如何删除一条工作经历？**
A: 在该经历卡片右上角点击"×"删除按钮。

**Q: 如何调整编辑器面板的宽度？**
A: 编辑器面板和预览区之间有一条竖线，鼠标悬停时会变宽，按住拖拽即可调整宽度。
